🚀 القسم 2: خطوات الرفع على Vercel
Section 2: Steps to Deploy on Vercel
═══════════════════════════════════════════════════════════════

الطريقة 1️⃣: الرفع عبر GitHub (مستحسن):
────────────────────────────────────────────

1. اذهب إلى GitHub.com وسجل دخول

2. أنشئ repository جديد:
   - اسم Repository: aystools-activation
   - اجعله Public أو Private
   - لا تضف README

3. ارفع ملفات vercel-website إلى الـ repository:
   ```bash
   cd vercel-website
   git init
   git add .
   git commit -m "Initial commit"
   git remote add origin https://github.com/YOUR_USERNAME/aystools-activation.git
   git push -u origin main
   ```

4. اذهب إلى Vercel.com وسجل دخول

5. اضغط "New Project"

6. اربط حسابك مع GitHub (إذا لم تفعل ذلك)

7. اختر repository: aystools-activation

8. اضغط "Deploy"

9. ✅ انتهى! ستحصل على رابط مثل:
   https://aystools-activation.vercel.app


الطريقة 2️⃣: الرفع المباشر عبر Vercel CLI:
────────────────────────────────────────────

1. ثبّت Vercel CLI:
   ```bash
   npm install -g vercel
   ```

2. سجل دخول:
   ```bash
   vercel login
   ```

3. انتقل لمجلد vercel-website:
   ```bash
   cd vercel-website
   ```

4. ارفع المشروع:
   ```bash
   vercel
   ```

5. اتبع التعليمات على الشاشة

6. ✅ انتهى! ستحصل على رابط النشر


الطريقة 3️⃣: الرفع عبر واجهة Vercel:
────────────────────────────────────────────

1. اذهب إلى Vercel.com

2. اضغط "New Project"

3. اختر "Import Git Repository" أو "Import from GitHub"

4. ارفع ملفات المشروع يدوياً

5. اضغط "Deploy"

🔧 القسم 3: تحديث رابط API في الأداة
Section 3: Update API URL in Plugin
═══════════════════════════════════════════════════════════════

بعد رفع الموقع، ستحصل على رابط مثل:
https://aystools-activation.vercel.app

⚠️ مهم جداً: عليك تحديث الرابط في الملفات التالية:

1️⃣  في ملف: online_license_manager.py
────────────────────────────────────────────

افتح: AySTools/backend/online_license_manager.py

ابحث عن السطر (حوالي 13-14):
```python
self.api_url = "https://wlcteest.vercel.app/api/activate"
self.activation_page = "https://wlcteest.vercel.app/activate.html"
```

غيّره إلى:
```python
self.api_url = "https://YOUR-PROJECT.vercel.app/api/activate"
self.activation_page = "https://YOUR-PROJECT.vercel.app/activate.html"
```

استبدل YOUR-PROJECT برابطك الفعلي!


2️⃣  في ملف: activate.html
────────────────────────────────────────────

افتح: vercel-website/activate.html

ابحث عن السطر (حوالي 195):
```javascript
const API_URL = 'https://wlcteest.vercel.app/api/activate';
```

غيّره إلى:
```javascript
const API_URL = 'https://YOUR-PROJECT.vercel.app/api/activate';
```


3️⃣  أعد رفع الملفات بعد التعديل:
────────────────────────────────────────────

- إذا استخدمت GitHub: اعمل push للتغييرات
- إذا استخدمت Vercel CLI: اكتب `vercel --prod`

🧪 القسم 4: اختبار النظام
Section 4: Testing the System
═══════════════════════════════════════════════════════════════

1️⃣  اختبار صفحة التفعيل:
────────────────────────────────────────────

افتح في المتصفح:
https://YOUR-PROJECT.vercel.app/activate.html?device_id=TEST123

✅ يجب أن تظهر صفحة التفعيل بشكل صحيح


2️⃣  اختبار API:
────────────────────────────────────────────

استخدم Postman أو curl:

```bash
curl -X POST https://YOUR-PROJECT.vercel.app/api/activate \
  -H "Content-Type: application/json" \
  -d '{"license_key":"AYSTOOLS-2025-PREMIUM","device_id":"TEST123"}'
```

✅ يجب أن تحصل على:
```json
{
  "success": true,
  "message": "تم التفعيل بنجاح"
}
```


3️⃣  اختبار من الأداة:
────────────────────────────────────────────

1. ضع الملفات المحدثة في Millennium
2. أعد تشغيل Steam
3. يجب أن تفتح صفحة التفعيل تلقائياً
4. أدخل مفتاح: AYSTOOLS-2025-PREMIUM
5. ✅ يجب أن يظهر إشعار "تم التفعيل بنجاح"

🐛 القسم 5: حل المشاكل
Section 5: Troubleshooting
═══════════════════════════════════════════════════════════════

❌ المشكلة: CORS Error
────────────────────────────────────────────
✅ الحل:
   - تأكد من وجود ملف vercel.json
   - تأكد من إعدادات CORS في api/activate.js


❌ المشكلة: 404 Not Found
────────────────────────────────────────────
✅ الحل:
   - تأكد من رفع جميع الملفات
   - تأكد من المسارات في vercel.json


❌ المشكلة: صفحة التفعيل لا تفتح
────────────────────────────────────────────
✅ الحل:
   - تأكد من تحديث الرابط في online_license_manager.py
   - تأكد من صلاحيات فتح المتصفح


❌ المشكلة: المفتاح الصحيح لا يعمل
────────────────────────────────────────────
✅ الحل:
   - تحقق من المفاتيح في api/activate.js (سطر 10)
   - تأكد من كتابة المفتاح بشكل صحيح (UPPERCASE)


❌ المشكلة: لا يكتشف التفعيل
────────────────────────────────────────────
✅ الحل:
   - تحقق من الـ logs في Millennium
   - تأكد من عمل التحقق الدوري (كل 5 ثوانٍ)
   - جرّب إعادة تحميل الصفحة (F5)

    'VIP-ACCESS-UNLIMITED'

const VALID_KEYS = [

    'CUSTOM-KEY-123'
